# Ticket

## Request

Implement a CLI in the repository root based on the `reference-repo/` implementation that allows taking individual Excalidraw files and creating animations from them.

The requested capability set includes:

- exporting only specific frames from an Excalidraw file and animating them
- changing the order of animations instead of relying only on original draw order
- getting the elements, reordering them, and controlling animation sequencing
- creating GIF and MP4 outputs
- creating PowerPoint slides with the animated elements included

Constraints noted in the request:

- do not edit the `reference-repo/` code directly
- copy or reimplement the relevant functionality at the repository root
